# Track: Scottish Scraper Engine

## Overview
Implement a robust but simple scraper to ingest news from key Scottish outlets.

## Tasks
- [ ] Initialize Python environment and dependencies.
- [ ] Define SQLite schema for staged stories.
- [ ] Implement RSS ingestion for BBC Scotland, The Scotsman, and The Herald.
- [ ] Implement text extraction (trafilatura) and metadata harvesting (author, date, lead image).
- [ ] Create a CLI runner to trigger the scrape.

## Success Criteria
- [ ] Scraper fetches at least 5 stories from at least 2 sources.
- [ ] Staging database contains text, author, date, and image URL for each story.
